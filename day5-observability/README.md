# Day 5 — Observability

Topic for today: **Observability**. Both AWS apps and LLM apps are basically opaque without it, and most tutorials hand-wave it. Doing it properly so we can actually debug what we ship.

## Topics

1. CloudWatch Logs + Logs Insights — structured logging and useful queries
2. CloudWatch Metrics + alarms — custom metrics, dashboards, what's worth alerting on
3. AWS X-Ray — distributed tracing, service map, sampling
4. LangSmith / OpenTelemetry for agents — token, tool-call, and prompt traces
5. Cost and latency observability for LLM apps — what to track per request
